# Self Healing CI/CD

This is a self-healing CI/CD pipeline system that automatically detects deployment failures and performs intelligent recovery actions including automatic rollbacks, retries, and alerting.

# Features

- Automatic Rollback: Intelligently rolls back failed deployments to last known good version
- Smart Retry Logic: Configurable retry mechanism with exponential backoff
- Real-time Monitoring: Prometheus + Grafana dashboards for complete observability
- Health Checks: Continuous monitoring of deployment health and pod status
